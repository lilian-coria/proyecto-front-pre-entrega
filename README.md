# Hilos y Encantos

Este proyecto es una página web desarrollada para el curso de Frontend. El sitio está dedicado a la venta de productos artesanales realizados en crochet y tejidos, ofreciendo una experiencia visual atractiva y moderna.

## Características principales
- **Página principal**: Presenta productos destacados con imágenes y precios.
- **Sección Nosotros**: Explica la historia y propósito del emprendimiento, acompañada de un video ilustrativo.
- **Formulario de contacto**: Permite a los usuarios enviar consultas de manera sencilla y rápida.
- **Carrito**: Opción para agregar productos y simular una compra.
- **Footer**: Información de contacto, redes sociales y derechos reservados.
- **Diseño responsive**: Adaptado para dispositivos móviles y escritorio.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- [Font Awesome](https://fontawesome.com/) para íconos

## Cómo usar
1. Clona el repositorio o descarga los archivos.
2. Abre `index.html` en tu navegador preferido.
3. Navega por las secciones y prueba el formulario de contacto.
